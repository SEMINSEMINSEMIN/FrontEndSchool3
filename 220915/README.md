## flex란?
내부 디스플레이 타입 중 하나
* 외부 디스플레이 타입 vs 내부 디스플레이 타입
    * 외부 디스플레이 타입(inline, block, inline-block): 부모와 형제간의 박스 배치
    * 내부 디스플레이 타입(flex, grid 등): 내부 자식 박스 배치에 영향을 미침
